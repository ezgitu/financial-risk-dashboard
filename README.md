# 📊 Financial Risk Dashboard

## Overview

This project is an interactive **financial risk dashboard** built with **Streamlit**.
Users can upload a CSV file containing financial data and instantly view risk scores, visual analytics, and personalized recommendations.

## Features

* Upload CSV data
* Analyze income, expenses, savings, and debt
* Calculate financial risk scores
* Display interactive visual insights
* Generate personalized recommendations
* Store processed data in SQLite

## Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib
* SQLite

## Installation

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

## Expected CSV Format

```csv
gelir,gider,tasarruf,borc
25000,12000,5000,3000
18000,15000,1000,7000
30000,14000,8000,2000
```
